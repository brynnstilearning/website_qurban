# Website Qurban

Sistem Informasi Qurban berbasis PHP Native dan MySQL.

## Fitur

- Login Multi User
- Manajemen Data Warga
- Manajemen Data Panitia
- Data Hewan Qurban
- Keuangan Qurban
- Generate QR Code
- Laporan

## Teknologi

- PHP Native
- MySQL
- Bootstrap
- PHP QR Code

## Instalasi

1. Clone repository
2. Import database
3. Atur konfigurasi database pada config/koneksi.php
4. Jalankan melalui Laragon/XAMPP
